# 概览
语音消息服务（UVMS）为企业用户提供方便快捷、安全可靠、高接通率、低延时、性能稳定的优质语音通信服务，包括语音验证码、语音通知等消息类服务，同时支持在线选号，高效触达用户的同时提升企业形象。
